# P0 Bugfixes Implementation Plan

> **For agentic workers:** Fix P0 bugs one by one, verify after each fix.

**Goal:** Fix 5 critical bugs that can cause crashes, deadlocks, or incorrect behavior.

---

## Task 1: Fix sd_storage mutex deadlock

**Problem:** `sd_storage_write_raw()` holds `s_buf_mutex`, then calls `switch_buffer()` which tries to acquire `s_buf_mutex` again → FreeRTOS non-recursive mutex deadlock.

**Fix:** Create `switch_buffer_nolock()` variant.

**Files:**
- Modify: `components/sd_storage/sd_storage.c:190-200, 435-440`

**Steps:**
1. Read switch_buffer() and sd_storage_write_raw() to understand the lock flow
2. Rename current switch_buffer to switch_buffer_nolock (remove internal lock)
3. Update all callers of switch_buffer_nolock (sd_storage_write_raw already holds lock)
4. Keep switch_buffer wrapper that acquires lock + calls nolock variant (for other callers)
5. Build and verify

---

## Task 2: Fix main.c direct s_system_state write

**Problem:** `main.c:1130` does `s_system_state = STATE_BLE_PAIRING` bypassing spinlock.

**Fix:** Replace with `system_set_state(STATE_BLE_PAIRING)`.

**Files:**
- Modify: `main/main.c:1130`

## Task 3: Fix divide-by-zero in main.c

**Files:**
- Modify: `main/main.c:257`
